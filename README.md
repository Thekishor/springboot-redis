## API Endpoints

### Product
- `POST /product/create` → Create a new product
- `GET /product/getById/{id}` → Get product by ID
- `GET /product/getAll` → Get all products
- `PUT /product/updateById/{id}` → Update product
- `DELETE /product/deleteById/{id}` → Delete product

  port: 9012
